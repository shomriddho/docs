আমরা যেকোনো আকারের অবদান এবং যেকোনো দক্ষতার কন্ট্রিবিউটরদের স্বাগত জানাই। একটি ওপেন-সোর্স প্রকল্প হিসেবে, আমরা আমাদের কন্ট্রিবিউটরদের প্রতিদানে বিশ্বাস করি। আমরা PR-এ গাইডেন্স, টেকনিক্যাল রাইটিং এবং যেকোনো ফিচার আইডিয়াকে বাস্তব রূপ দিতে সাহায্য করতে প্রস্তুত।

আরও গভীরভাবে যুক্ত হতে চান? আমাদের [গভর্ন্যান্স ডকুমেন্টেশন](https://github.com/withastro/.github/blob/main/GOVERNANCE.md) দেখুন যেখানে বিভিন্ন ভূমিকা, মেইনটেইনার মনোনয়ন প্রক্রিয়া, কোড রিভিউ এবং কোড অব কনডাক্ট প্রয়োগের বিস্তারিত বর্ণনা রয়েছে।

## অবদান রাখার উপায়

### প্রজেক্ট রিপোজিটরি সমূহ

অ্যাস্ট্রো প্রকল্পে অবদান রাখার অনেক উপায় রয়েছে! প্রতিটি অ্যাস্ট্রো রিপোজিটরির রুটে একটি `CONTRIBUTING.md` সহ README থাকে।

[Astro-এর GitHub প্রোফাইল](https://github.com/withastro) এ যান এবং নিচের রিপোজিটরিগুলি দেখুন:

- [মূল অ্যাস্ট্রো কোডবেস](https://github.com/withastro/astro), যার মধ্যে রয়েছে অফিসিয়াল ইন্টিগ্রেশন ও স্টার্টার টেমপ্লেট।

- [Astro ডক্স](https://github.com/withastro/docs), একটি সম্পূর্ণ অ্যাস্ট্রো ওয়েবসাইট! এখানে কন্টেন্ট লেখার পাশাপাশি a11y, CSS, UI, এবং UX সম্পর্কিত অ্যাস্ট্রো কোডেও অবদান রাখা যায়। আমরা একাধিক ভাষায় ডকুমেন্টেশন প্রকাশ করি, তাই অনুবাদেও সহায়তা প্রয়োজন।

- [Astro কম্পাইলার](https://github.com/withastro/compiler), যা Go ভাষায় লেখা এবং WASM হিসেবে বিতরণ করা হয়।

- অ্যাস্ট্রোর [ভাষা সরঞ্জাম](https://github.com/withastro/language-tools), `.astro` ফাইলের জন্য প্রয়োজনীয় এডিটর টুলিং।

- [Starlight](https://github.com/withastro/starlight), অ্যাস্ট্রোর অফিসিয়াল ডকুমেন্টেশন ফ্রেমওয়ার্ক।

- [Astro রোডম্যাপ](https://github.com/withastro/roadmap), যেখানে অ্যাস্ট্রোর ভবিষ্যৎ গঠিত হয়! এখানে আপনি আইডিয়া, সাজেশন, এবং আনুষ্ঠানিক RFC প্রস্তাব দিতে পারেন।

### কন্ট্রিবিউশনের ধরন

আপনার নিজস্ব কোড বা কনটেন্ট দেওয়ার পাশাপাশি PR-এ রিভিউ কমেন্ট করা, GitHub Issues বা Discussions-এ আইডিয়া যোগ করা, এবং "Pinned" ইস্যুতে অংশগ্রহণ করাও বড় অবদান।

প্রতিটি PR, বিশেষ করে অনুবাদ PR, রিভিউয়ের প্রয়োজন হয়! PR রিভিউ করা এবং মন্তব্য, সাজেশন, বা একটি "LGTM!" ("Looks Good To Me!") দেওয়া শুরু করার জন্য চমৎকার উপায়।

আমাদের একটি সক্রিয় [Discord](https://astro.build/chat) কমিউনিটিও রয়েছে! নতুন সদস্যদের স্বাগত জানানো, সাহায্যের প্রশ্নের উত্তর দেওয়া, এবং অ্যাস্ট্রো দিয়ে তৈরি কিছু শেয়ার করাও গুরুত্বপূর্ণ কন্ট্রিবিউশন। শুধু GitHub নয়, ব্লগ পোস্ট, ভিডিও, কনফারেন্স টক, এবং কমিউনিটির স্বাস্থ্য রক্ষা করেও আপনি অবদান রাখতে পারেন।

## ডকস-এ অবদান রাখা

Astro Docs-এ অবদান রাখতে আপনাকে সহায়তা করার জন্য আমাদের বেশ কয়েকটি গাইড রয়েছে।

আপনার যদি এটি হয় প্রথম ওপেন সোর্স কন্ট্রিবিউশন, অথবা আপনি নতুন কোনো Astro ফিচারের জন্য ডকস লিখছেন, অথবা একজন অভিজ্ঞ অনুবাদক হিসেবে পরবর্তী পৃষ্ঠাটি খুঁজছেন — আমাদের কাছে আপনার জন্য সহায়তা রয়েছে।

আমাদের নির্ধারিত সাইট [*Astro Docs* ডকস](https://contribute.docs.astro.build) দেখুন। এখানে আপনি পাবেন তথ্য কীভাবে টাইপো ঠিক করা, লেখা, অনুবাদ, ফিচার তৈরি, এমনকি PR রিভিউয়ার হিসেবেও অবদান রাখা যায়।
